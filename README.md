## Data
This analysis uses data from the show "The Great British Bake Off" to look at contestants' perfomance in the Technical Challenge and how it relates to their performance on the show overall.
The dataset includes Series 1-10 of the show, and is compiled in the R package [{bakeoff}](https://bakeoff.netlify.app/) by Alison Hill.
